## Value-List

| Token      | Syntax   | Name       |
| :--------: | :------: | :--------: |
| `{` .. `}` | _.._ _?_ | Value-List |

```minimaL
{ 0 1 2 3 } 2 ** { A B C D } :
```

- Answers
  1. [_A_] 0
  2. [_B_] 2
  3. [_C_] 4
  4. [_D_] 9
