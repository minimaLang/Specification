# Syntax

| Syntax               |
| :------------------: |
| <1> `'`              |
| <1..*>_Identifier_   |
| (<0..1>`Dash`)       |
| (<0..*>_Identifier_) |
| (<0..1>`Underscore`) |

1. Symbols enclosed in round brackets are optional.
2. One number enclosed in `<` `>` shows the mandatory length.
3. The Symbol `..` between numbers indicates the permitted frequency.
   - `..*` indicates an unlimited frequency.
